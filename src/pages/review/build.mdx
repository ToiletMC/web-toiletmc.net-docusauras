# 👷‍♀️ 建筑审核

**审核管理员**：SmerryBeta  
**审核时间**：每周都会审核，通过的建筑可以继续审核，但是要在原来的建筑中加以改进。  
**称号奖励**：建筑通过审核后，系统会下发称号可以 `/tags` 查看。  
**提交审核**：游戏内联系审核管理员，或提交问卷 https://wj.qq.com/s2/14613134/9417  
**注意事项**：所有建筑都必须有**明确的建筑主题**且**独立**完成建筑作品，建筑作品**不可以完全抄袭**。

Q1：什么是明确的建筑主题？  
堡垒，中世纪宫殿，中国风建筑等，不知道你的建筑风格是什么？[点击查看](https://www.bilibili.com/read/cv15967515/)。

Q2：我为什么没有收到称号？  
因为是人工发放称号，所以不会及时给到称号，没拿到称号也不要着急，等管理上线会下发的。

:::
