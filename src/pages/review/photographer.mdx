# 📸 摄影师称号

**提交审核**：https://wj.qq.com/s2/14628127/w08p

如果你有在游戏里发现美的心，请用截图键记录下来。要求运用光影/渲染，截图数仅接受 **5 张**。一旦接受可以获得摄影师称号及奖励，且图片有机会登上官网或附加在宣传帖中。内容不限，但请注意构图。

鉴于很多人不知道这个标准在哪里…这些图是会出现在官网和宣传帖的，如果不是很吸引人就骗不了新人来玩。所以有几个拍摄注意的事项我说一下，以免白做工：

1. FOV （视场角）最好调回默认。
2. 材质包最好不要有方块边界。
3. 光影最好不要用画面昏黄一片的（很多光影的调校可能用了 HDR 或者怎样，在 SDR 屏幕上显示就很糟糕）。
4. 注意构图，知道自己在拍什么，就和在现实里拍照一样。
5. 视角如果受限，可以尝试开启 tweakroo 的灵魂出窍。
6. 如果是宏大建筑，可以尝试使用 Chunky 直接渲染地图。
