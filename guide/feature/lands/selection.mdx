关于我们的服务器圈地系统 `/land`，我相信在其他服务器使用的 `/res` 圈地系统的玩家们难免对 `/land` 圈地系统感到陌生。对此，我将为各位新来到我们服务器的玩家们，以及不熟悉 `/land` 圈地系统的长期玩家提供最基本的 `/land` 指令圈地教程。在此基础上，你还可以通过指令 `/land help` 来获得更多有关此圈地系统的知识。

:::tip 小贴士
请合理使用领地插件保护你的领地不受他人破坏！
:::

## 指令 /land edit [领地名称]

作为操作声明的指令存在，比方说在委托他人做某些事情之前必须告诉对方你需要对某个东西做什么，如果没有对对方说明需要做什么，对方就不知道你要做什么一样缺乏方向。

在指令 `/land edit` 也是如此，`/land edit` 会默认对你所处的领地执行选定操作。当你不在目标领地时，你需要使用 `/land edit [领地名称]` 即可进行远程操作。

示例如下：我想要在 RGO 领地周围占领领地使用的是首先执行 `/land edit RGO`，其次执行 `/land claim` 。

![](/img/guide/lands/Untitled1.png)

## 指令 /land menu

如图所示，点击你需要编辑的领地进入该领地的菜单当中。

![](/img/guide/lands/Untitled3.png)

你可以在里面完成各种关于玩家角色、领地、欢迎语、更改领地名称、等方面的设置。

## 指令 /land delete

实例如下：假设我要删除领地 RGO  
`/land edit RGO` ←RGO 可视作[领地名称]，在执行 `/land delete` 前必须输入该指令！

![](/img/guide/lands/Untitled4.png)

## 指令 /land selection 和 assign

在使用 `/land edit [领地名称]` 选定操作领地后，在这里，你需要先执行 `/land selection`，你将得到一个金锄头在手上，如果切走它，金锄头将消失，你需要重新执行 `/land selection`。

左键选中第一个点，右键选中第二个点，使用 `/land assign [你想要分配到的子领地]`。
此外，你也可以在执行 `/land selection` 的相关操作后，使用 `/land claim` 大量占领选中的区块。

## 指令 /land trust [玩家名称] 和 /land accept [领地名称]

在使用 `/land edit [领地名称]` 选定操作领地后，你可以通过 `/land trust [玩家]` 或者 `/land menu` 菜单向玩家[玩家]发送领地邀请。

在该玩家通过执行 `/land accept [你的领地名称]` 接受邀请后，你可以在 `/land menu` 菜单中编辑该玩家的角色权限组。
