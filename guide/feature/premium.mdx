## 操作方法

执行此指令即可：

```plain
/premium
```

:::caution

- **非正版请勿操作**，会导致无法进入游戏！
- 正版验证会**丢失**你正在游戏的数据！
- 如果最近才购入了正版，请联系管理员协助您从离线账号迁移数据！

_🚧 自动迁移系统正在路上…_
:::

## 正版玩家权益

- 正版称号（自动发放，需手动开启，称号会显示在名字前面）
- 在线皮肤（由服务器显示皮肤和披风，且无需任何 Mod）
- 免密登陆（加入服务器不再需要密码登陆）
- 修改玩家名保留游戏数据（[minecraft.net](https://www.minecraft.net/zh-hans/msaprofile/mygames/editprofile) 修改玩家名后再次正版认证即可）

## 撤销正版验证

```plain
/cracked
```

:::caution

- 执行命令后，您会被系统标记为离线模式，将在下一次进入服务器时失去正版权限。
- 如果您忘记了验证前的密码，请联系管理员重置密码！
  :::
