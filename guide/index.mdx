# 📃 介绍

🙂 欢迎来到厕所总部！作为一个创立于 2019 年 8 月 15 日的 Minecraft 服务器，我们一直致力于提供最好的游戏体验。

⬆️ 我们坚持带档升级，因为我们相信每个玩家的存档是宝贵的记忆。在许多服务器中，版本更新会导致存档被删除，但在厕所总部，我们会保留存档，并确保它们与新版本兼容。

✅ 我们的玩法是稳定的。相较于其他服务器，在每次版本更新时，我们不会频繁更改服务器的功能和删除插件，因为我们深知这可能会破坏玩家们的游戏体验。相反，我们精心设计每个功能，并确保它们能够在未来的版本更新中继续存在，让你可以享受到稳定的游戏玩法。

🥳 在这里，你可以找到一个充满活力、友好和融洽的社区。我们的管理员和志愿者们非常乐意帮助新玩家，并确保每个人都能拥有愉快的游戏体验。如果你正在寻找一个稳定、有趣、友好的 Minecraft 服务器，那么请加入我们吧！

## 如何游玩？

服务器版本：1.20.4  
我们允许使用 1.9.x - 最新 版本的客户端加入游戏。  
但为了更好的游戏体验，我们鼓励你使用最新版本加入游戏。

以下内容已被 [Paper](https://papermc.io/software/paper) 修复，我们也不会开启：

- 破基岩
- 任何刷物品的 BUG

以下内容已被关闭，我们通常不会开启：

- 所有世界的苦力怕炸毁方块
- 所有世界的末影人拿起蘑菇
- 生存世界的火势蔓延

以下内容已被启用，滥用将会被管理员封禁：

- 活塞复制：包括 TNT 复制、地毯复制

### 基岩版

:::tip

- 使用基岩版客户端游玩服务器需要登录 Xbox 账户。
- 游玩前推荐进行账号链接，实现 JAVA-基岩 玩的是同一个账号。
  :::

```
29239
```

### 账号链接指南

#### 对于拥有 JAVA 正版账户的玩家

- 请优先前往 https://link.geysermc.org 进行账号链接。

#### 对于使用离线模式或网络不佳玩家

1. 需要在本服先使用 JAVA 版客户端登录，注册并登陆后输入指令：`/linkaccount Xbox玩家代号`
2. 使用基岩版加入服务器，然后输入指令：`/linkaccount JAVA版游戏名 验证码`
3. 恭喜你已经完成了账号链接，尽情游戏吧！

:::caution
账号链接完成后，JAVA 版账号将被设置为主要游玩账号，请将基岩版账号内的背包物品、领地以及其他财产转移后再进行账号链接！（如果操作失误，请使用指令取消链接：`/unlinkaccount`）
:::

:::tip

- Xbox 玩家代号中包含空格，需要使用英文状态下的引号引起来，例如：`/linkaccount "Little Pig"`
- 账号链接后，如果您后悔了，可以随时使用指令取消账号链接：`/unlinkaccount`
  :::
