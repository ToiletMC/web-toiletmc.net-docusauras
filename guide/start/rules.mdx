# 📜 服务器规则

## 尊重他人

尊重他人意味着不要给服务器玩家带来负面的游戏体验。

- 禁止发表歧视性言论：如对他人的种族、性别、年龄、学历、地域或其他个人特质进行负面评价。
- 禁止故意骚扰、欺凌、辱骂、人身攻击、发表仇恨性言论或煽动、挑拨他人进行。
- 禁止在未经他人同意的情况下，公开其个人信息。
- 禁止刷屏：重复在聊天框输入类似的内容。
- 禁止发表裸露以及露骨色情内容。
- 禁止发表恶意或欺骗性内容：例如恶意软件、病毒、破坏性指令、商业诈骗等。
- 禁止发表虚假信息，对他人进行诬陷、诽谤。
- 禁止发表极端负面的内容，例如自残、自杀等伤害自己或他人的内容。如您被相关心理问题困扰，建议寻求当地医疗机构的帮助。

触犯本节条例的，视情节严重予以禁言、投入监狱或封禁账号处罚。

## 请勿卡服

为了给每一位玩家提供良好的游戏体验，服务器不欢迎有大量占用服务器资源的发展行为。

卡服行为包括但不限于：

- 囤积大量实体
- 建造卡服机器
- 建造 8bit 红石计算机
- 超大规模的生产线、刷怪塔
- 包含超多货运节点的粘液基地

请合理使用货运节点。我们建议您不要同时运行所有的货运节点。不同的物品请错峰生产，不要一次同时生产所有物品。为了将货运节点数量控制在服务器能力负载范围之内，我们建议您将同时使用的货运节点数量控制在 30~50 个，不要滥用货运节点，不要一味地追求效率。

![](/img/guide/slimefun_lag.png)

:::warning

- 由玩家的任何行为或其所在地引起的服务器卡顿，风机委员有权要求其离开或停止工作，若玩家在风纪委员提醒后依旧选择无作为风纪委员有权对其处以 1 分钟 - 7 天 的监禁。
- 由玩家故意造成的卡服（由风纪判断）将处以 3 天以上的监禁且罚没 10%的个人资产，情节严重者将封号处理。

:::

## 请勿破坏

每一座建筑都凝聚着建设者的心血，服务器对破坏他人建筑的行为持零容忍态度。

破坏行为包括但不限于：

- 挖掘
- 用各种方式引发的爆炸
- 使用活塞推拉
- 使用水或岩浆冲毁
- 纵火烧毁等方式使建筑遭到损创

破坏行为发生后能当场将被破坏建筑恢复原貌的，可以免于处罚。

:::warning

- 恶意破坏达到四级损毁，处拘禁 24 小时，罚没 25%个人资产。
- 恶意破坏达到三级损毁，处拘禁三天，罚没 50%个人资产。
- 恶意破坏达到二级损毁，处拘禁七天，罚没 75%个人资产。
- 恶意破坏达到一级损毁，处以 BAN IP 的处罚。
- 以上行为累积达到 3 次者处以 banIP 处罚。

:::

## 请勿作弊

我们希望我们的玩家都在公平地玩游戏，即不利用任何作弊手段或漏洞来获得优势。作弊、外挂、使用 Mod 和第三方工具是不被允许的。在游戏中利用漏洞也同样不会被容忍。如果我们侦测到某玩家正在以任何方式在游戏中作弊，我们将会立即封禁该玩家的账户。

会使您的游戏账户被禁止的行为包括但不限于：

- 侵入或修改游戏
- 利用游戏漏洞
- 使用未经授权的第三方软件来修改或干预游戏

:::warning

- 利用服务器市场、商店、特殊道具等漏洞盈利者，处 14 日监禁，没收违法所得，并罚没个人 50%的资产。
- 利用服务器市场、商店、特殊道具等漏洞盈利且隐藏 BUG 实情不汇报者，处 60 日监禁，罚没个人全部资产。
- 以上罪行累犯两次者，处以 BAN IP 的处罚。

:::

:::warning

- 使用不被服务器允许的程序（自己写的也算）插件或 Mod（矿透，杀戮光环等）处 14 日监禁，并罚没个人 75%的资产。
- 累计开挂两次者，处以 BAN IP 的处罚。

:::

## 请勿偷盗或占用他人物品

偷盗行为包括但不限于：

- 不经允许情况下拿取不属于自己的物品
- 使用各类爆炸物对他人容器进行爆破以盗取内容物
- 偷取他人的牲畜、宠物等

有偷盗行为的，投入监狱限制行动

占用他人物品包括但不限于：

- 跟他人借用物品后拒绝归还等

:::warning

- 经发现后主动归还，态度较好的处三日拘禁，可酌情继续减刑。
- 经发现后不主动归还，处七日拘禁以及罚没个人 10%的马桶币。
- 经发现后不主动归还，且为自己罪行百般开脱甚至与风纪委员对线，处十日拘禁以及罚没个人 25%的马桶币。
- 以上罪行多次触犯屡教不改者，除第一条外，罚款金额随犯罪次数上调（2：10%——25%——40% 3：25%——40%——50%）。第一条按照犯罪次数处罚向 2,3 顺延。以上罪行累犯四次者，处以 BAN IP 的处罚。

:::

## 请勿恶意击杀

恶意击杀包括但不限于：

- 直接击杀
- 制造陷阱陷害
- 使用爆炸物炸死他人及他人的牲畜、宠物，或令他人及他人的牲畜、宠物掉入虚空。
- 在非 PVP 场地（图书馆、战壕等）恶意攻击其他玩家

有恶意击杀行为，并被被击杀人举报，且不能取得被击杀人原谅的，予以投入监狱的处罚。

双方认可的 PVP 行为，或已取得被击杀人谅解的，可以免于处罚。

:::warning

- 恶意攻击一经举报，处拘禁半小时。
- 恶意攻击导致玩家死亡，一经举报，处拘禁一小时。
- 以上罪行第二次触犯均处两小时监禁，此后每触犯一次拘禁时间累加一小时，封顶为 24 小时，若屡教不改则会以封号处理。
- 在对其他玩家进行造谣，辱骂，挑衅等言语暴力时对当事人造成影响，一经证实将对参与者处以监狱 1-7 天，情节严重者将处以封号。

:::

## 请勿创作违反公序良俗的内容

我们支持玩家有权利自由地表达和创作自己的作品，但相应的，玩家也有权利不要被某些特定内容冒犯。这些内容包括：

- 不合适的像素画
- 不合适的书籍
- 不合适的用户名
- 不合适的装备名
- 除上述之外所有可自定义文字的场景
- 不合适的定义包括但不限于：色情、暴力、反动、血腥等负面内容

触犯本节条例的，予以收回拆除并处罚金或投入监狱。
