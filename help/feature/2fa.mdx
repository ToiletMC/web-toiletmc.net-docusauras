尽管强密码能够保证账号安全，但由于密码的复杂度不够，仍然存在被暴力破解的风险。

因此，厕所总部为玩家提供了 2FA (双因素身份验证) 功能，以进一步保证账号的安全。

## 启用 2FA

:::tip
如果您是正版用户，则可以使用 Microsoft 提供的 2FA 功能，无需配置。
:::

1. 通过以下命令生成代码
  ```plain
  /2fa add
  你的代码是 ...，你可以使用 ... 来进行扫描
  ```
2. 在您的验证应用中输入代码，或点击链接扫描二维码
3. 通过以下命令验证并启用 2FA
  ```plain
  /2fa confirm <2FA 验证码>
  已成功启用双重认证
  ```

## 停用 2FA

通过以下命令停用 2FA
```plain
/2fa remove <2FA 验证码>
双重认证码已从您的账号上删除
```
