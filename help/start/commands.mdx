# ⌨️ 常用指令

- `/hi` 打开菜单
- `/wild` 随机传送（第一次免费）
- `/spawn` 回到主城
- `/tpa` 传送到另一玩家处
- `/tpahere` 请求玩家传送到你这
- `/ds` 打开系统商店
- `/shop` 玩家商店帮助
- `/msg` 私信玩家
- `/pay` 转款给其他玩家
- `/kit` 领取礼包
- `/tags` 佩戴已有称号
- `/list` 在线者一览
- `/baltop` 马桶币排行榜
- `/playtimetop` 在线时间排行榜
- `/slimemap` 查看史莱姆区块

:::info 仅限风纪委员
- `/jail` 监禁
- `/unjail` 解除监禁
- `/mute` 禁言
- `/unmute` 解除禁言
:::
